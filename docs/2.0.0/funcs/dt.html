dt

New in 2.1.0

Description

dt is a built-in variable which represents the time the last frame took in microseconds as an integer.

Example

import agan

class game():
	def create(self):
		return self
	#
	def update(self):
		agan.log(agan.dt)
	#
	def close(self):
		pass
