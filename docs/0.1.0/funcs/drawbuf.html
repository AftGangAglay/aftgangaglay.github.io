drawbuf

Parameters

Parameter Type Description
handle nativeptr The vertex buffer handle to draw
primitive int The primitive to draw. Named constants are provided in the script support library
trans transform The transform to apply when drawing the buffer

Return Value

N/A

Description

Draws the vertex buffer data previously opened by mkvertbuf. Primitive types map to the OpenGL primitives.

Example

import agan
import aga

class game():
    def create(self):
        self.f = agan.mklargefile('vertices.raw')
        self.vbuf = agan.mkvertbuf(f)
        self.trans = aga.transform().create()
        return self
    #
    def update(self):
        agan.drawbuf(self.vbuf, aga.TRIANGLES, self.trans)
    #
    def close(self):
        agan.killvertbuf(self.vbuf)
        agan.killlargefile(self.f)
