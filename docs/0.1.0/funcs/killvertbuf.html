AftGangAglay Documentation

killvertbuf

Parameters

Parameter Type Description
handle nativeptr The vertex buffer handle to close

Return Value

N/A

Description

Closes a vertex buffer handle previously opened by mkvertbuf.

Example

import agan
import aga

class game():
	def create(self):
		self.f = agan.mklargefile('vertices.raw')
		self.vbuf = agan.mkvertbuf(f)
		self.trans = aga.transform().create()
		return self
	#
	def update(self):
		agan.drawbuf(self.vbuf, aga.TRIANGLES, self.trans)
	#
	def close(self):
		agan.killvertbuf(self.vbuf)
		agan.killlargefile(self.f)
