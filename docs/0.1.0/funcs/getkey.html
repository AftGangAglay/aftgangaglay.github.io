getkey

Parameters

Parameter Type Description
keycode int The code of the key whose state should be read. Available values are named in the script support library.

Return Value

Type Description
int The state of the requested key - True for held, False for released.

Description

Fetches the current state of the key named by keycode - held or released. Keycodes refer to physical keys and not characters, so modifiers are reported the same as others. If the requested keycode is outside of the keymap bounds, None is instead returned.

Example

import aga

class game():
    def create(self):
        return self
    #
    def update(self):
        if(aga.getkey(aga.KEY_w)):
            aga.log("The W key is held!")
        elif(aga.getkey(aga.KEY_l)):
            aga.log("The L key is held!")
        else:
            aga.log("Neither of my keys are held :(")
