mklargefile

Parameters

Parameter Type Description
path string The path to load the file from

Return Value

Type Description
nativeptr The loaded large file handle

Description

Opens a "large file strategy" handle - in practice this tries to mmap if the system supports it and otherwise uses the typical malloc + fread pattern.

Once no longer in use, handles should be closed using killlargefile.

Example

import agan

class game():
	def create(self):
		self.f = agan.mklargefile('test.txt')
		self.clip = agan.mkclip(f)
		return self
	#
	def update(self):
		pass
	#
	def close(self):
		agan.killclip(self.clip) # `clip' created from `f' and must die first.
		agan.killlargefile(self.f)
