AftGangAglay Documentation

killlargefile

Parameters

Parameter Type Description
handle nativeptr The large file handle to close

Return Value

N/A

Description

Closes a "large file strategy" handle previously opened by mklargefile. All objects created using the handle should be closed before calling killlargefile.

Example

import agan

class game():
	def create(self):
		self.f = agan.mklargefile('test.txt')
		self.clip = agan.mkclip(f)
		return self
	#
	def update(self):
		pass
	#
	def close(self):
		agan.killclip(self.clip) # `clip' created from `f' and must die first.
		agan.killlargefile(self.f)
