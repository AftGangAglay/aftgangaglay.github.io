AftGangAglay Documentation

clear

Parameters

Parameter Type Description
col float[] The clear color.

Return Value

N/A

Description

Clears the backbuffer and depth buffer with col, formatted as normalized RGBA.

Example

import agan

class game():
	def create(self):
		return self
	#
	def update(self):
		agan.clear([ 0.8, 0.4, 0.3, 1.0 ])
	#
	def close(self):
		pass
